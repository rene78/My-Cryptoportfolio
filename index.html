FAQ

xxWhat are the advantages of this portfolio app compared to others?
  • xx Not-for-profit. This app was solely written for my personal use and as an exercise. But I'd be glad if you'd find it useful, too. The source code can be accessed on Github.
  • xx Ad-free. No annoying ads, which slow down page load and impair user experience.
  • xx Cookie-free. No user information is saved to servers.
  • xx Hassle-free. Personal portfolio can be accessed via a unique URL. No login required.
  • xx Web app. No installation needed.
  • xx Responsive. Works on all devices.

xxIs it safe to enter my portfolio on this site?

As long as you don't share your personal portfolio URL your data should be safe. All portfolios are saved to the database in a completely anonymous fashion with no link to the user whatsoever. This web app does not collect any cookies (except third-party cookies from Disqus), does not have ads and is 100% open source. The source code can be viewed and changed on Github (My Cryptoportfolio).
